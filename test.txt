## MGF Staffing Estimation Methodology

To estimate if 9 people are enough, you need to convert the defined activities (SIT Assist, UAT, Parallel Run, Production Switch) into a quantifiable **time-per-application** metric and compare it against the total required effort in a given timeframe (Lot duration).

### 1\. Define Effort per Application (App)

First, estimate the average time (in person-days or FTE-days) the MGF team spends on all activities for **one application**. This is the most crucial step.

| **MGF Activity** | **Assumption/Scope** | **Estimated Effort (Days/App)** |
| :--- | :--- | :--- |
| **SIT Assistance** | Assisting NPW (External Vendor) with issue triage, coordination, JIRA updates, re-testing support until "Definition of Done" is met. | $T_{SIT}$ (e.g., 3-5 days) |
| **UAT Execution** | Independent UAT run by MGF, gathering App Team sign-off. | $T_{UAT}$ (e.g., 2-4 days) |
| **Parallel Run Support** | Monitoring/triage for the parallel run phase. | $T_{PR}$ (e.g., 1-2 days) |
| **Production Switch** | Cutover support. | $T_{P S}$ (e.g., 0.5-1 day) |
| **Total MGF Effort per App ($E_{App}$)** | | **$T_{SIT} + T_{UAT} + T_{PR} + T_{PS}$** (e.g., **6.5 - 12 days**) |

*Note: For a more granular estimate, this effort can be broken down by average **Jobs per App** (e.g., an App with 20 jobs takes more effort than one with 5).*

### 2\. Calculate Required Effort per Lot

Use the **Total MGF Effort per App** ($E_{App}$) and the number of applications in each lot ($N_{Lot}$) to find the total effort required per Lot.

$$Total\ Effort_{Lot} = E_{App} \times N_{Lot}$$

### 3\. Calculate Available Capacity per Lot

Determine the available working days for the current 9 MGF FTEs during the Lot's duration ($D_{Lot}$).

$$Available\ Capacity_{Lot} = 9\ FTEs \times D_{Lot}$$
*(Adjust for holidays and planned leave to get actual working days.)*

### 4\. Compare and Determine Need

The key is to compare the **Required Effort** against the **Available Capacity**.

| Lot | Apps ($N_{Lot}$) | Duration ($D_{Lot}$) | Available Capacity (e.g., 9 FTEs x $D_{Lot}$) | Required Effort (e.g., $N_{Lot}$ x $E_{App}$) | **Gap (Need)** |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Lot 1** | 3 | $\approx 20$ days | $\approx 180$ person-days | $3 \times E_{App}$ | Likely **sufficient** |
| **Lot 2** | 14 | $\approx 30$ days | $\approx 270$ person-days | $14 \times E_{App}$ | **Potential Gap** (If $E_{App}$ is high) |
| **Lot 7** | 21 | $\approx 45$ days | $\approx 405$ person-days | $21 \times E_{App}$ | **High Risk** (Likely requires $\uparrow$ FTEs) |

$$\text{Required Additional FTEs} = \frac{(Required\ Effort_{Lot} - Available\ Capacity_{Lot})}{D_{Lot}}$$
